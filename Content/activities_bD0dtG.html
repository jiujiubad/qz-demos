Megan_Ho 修改了作业 2 个月前
作业 - 作业一
1. 请阅读 Google 的效能指南以及PageSpeed Insights 的评分标准， “阅”

2&3. 用Lighthouse 侦测任一个你的网站，最好是已经部署的网站。请截图一张目前的 Performance 的分数报告
http://remote-jobs.herokuapp.com/


4. 请套用任何你学到的前端效能优化方式
新标准做法：async 和 defer 属性
app/views/layout/application.html.erb
-  <%= javascript_include_tag 'application', 'data-turbolinks-track': 'reload' %>
+  <%= javascript_include_tag 'application', 'data-turbolinks-track': 'reload', :defer => true %>
defer 的话，浏览器也不会阻挡 HTML 的渲染，但是当 JavaScript 下载完成后，会等 HTML 加载完成，才会执行 JavaScript。

5. 请分享你做了什么改进，以及截图一张最新的效能分数
