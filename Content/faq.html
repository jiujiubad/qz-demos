这个全栈营主要是教什么？你们的全栈定义是什么？

新生大学全栈营所培养的全栈工程师是这样的，我们希望经过这个课程洗礼的同学们，在毕业后能够从两个维度方面具备“全栈”的能力：

  • 能够独立策划、开发、运营开发互联网产品
  • 熟悉 Web 开发方面的全栈技能（ Rails / CSS / JavaScript / SQL / Server Deployment / …. )

可以参考 Quora 上这篇文章What are good ways to be a full stack developer，了解“全栈”在技术上的定义。

但是我们不想只在“技术”维度上做到这个定义，我们更希望在全方面的角度上做到“全栈”，也就是希望上完整套课程的人，甚至可以做到：独立技术产品创业这件事。

听起来很有野心，请问课程安排上如何达到这件事？

2016 年夏季起开办的全栈营 2 期“两个月线下版”以及各位看到的这个“全栈营线上版网站”，就是这门课最好的实践。各位看到的这个“全栈营”线上版。正是第一期的 3 个同学在毕业后 2 个月内（​​2016/10/12-2016/12/12 ) 开发的。

“两个月线下脱产版”的安排是这样的：

  • 前 3 周是全栈工程师的技术 / 普通版（也就是这3周，是教技术上独立开发的知识以及具体实践）。
  • 后 5 周是全栈工程师的多维黄金版（后面5周，从产品调研策划 Landing Page 开始、项目策划管理、小组高强度开发协作、架构重构效能调优、Onboarding 调整上线，完整模拟一个互联网公司优秀团队的产品开发实践）。

这样的高强度效果，成果甚至是外国的线下班做不到的。这样的效果也不仅仅只是文案上的夸饰，具体成品可以从这两期学生最后成果以及技术日记、周记看到展示与“直播”。

也因为这样的机会绝无仅有（十年经验老司机完整带练，而且一次只能带少量小组），所以线下脱产版定价才会这么昂贵。

也太高效了。许多线下培训班技术部分至少要花上 3 个月才上得完，成员毕业后功力也不行，你们是如何办到的？

其实几年前，在我们还没有投入技术培训这个产业之前，我们就发现许多技术培训班（不管是线上线下）的致命缺点了。

一般培训班的问题在于，很多人毕业学成之后，“基础”薄弱、无法“迁移”。

能够“迁移”是“求职录取”最重要的一环。也就是真实工作环境，往往是公司具体要求一个功能，开发者要在熟练基本工具的情况下，能够直接实现公司各种合理的功能。甚至是高级一点的开发者，要能够将市场的需求直接“落地”，与同事“协作”之后开发出来，并且运营调优。

绝大部分其他培训班的 90% 学员，很难在毕业后直接做到这一点。多半是超过培训课时的模板项目范畴，就不知道怎样做出来了。所以往往毕业后，还需要一定的时间“熟成”，才能找到“理想的工作”。

我们这个班绝对跟其他培训班不同，当初全栈营的培训宗旨在一开始就很明确：

  • 以同学毕业后直接可以独立开发实现功能为目的
  • 同学甚至要达到市场上的梦幻需求：能够将市场的需求直接“落地”，与同事高效率“协作”

再加上我们的首席教练是市场的十年老司机大牛，做过无数个项目，带过大大小小的工程团队，公司内部培训出几十个 Rails 程序员，连续创业过数次。熟悉各种做项目、开发、调优、运营的坑与套路，并知道如何引导新手闪坑达标。

所以我们才有办法设计出这套课程，让在学员短时间内就变身成市场上合格的程序员。别的培训班甚至是花了三个月才有办法达到我们三周的效果。

全栈营“线上版”与“线下版”具体差异是什么？

头两期的“线下版全栈营”是实验性的。席次少价格也贵，但成效非常好。后续我们陆续接到全国非常多同学：

  • 是否可以考虑到全国各地开班
  • 移植到线上班的可能性

我们决定将最新一期的全栈营，拆成全新的两种形式：

  • 线上 8 周版。 （也就是现在各位看到的这个版本）
  • 接续线下 5 周团队项目版。 （必须修毕线上版才能报名，费用另公布。）

线上 8 周版

  • 将会是“全栈营线下三周密集版”的展开版本。线上教学以及助教辅导，各地组成线上交流团体。
  • 学生每周将至少要投入 10-15 小时的学习、练习时间。
  • 将会学习到基础求职等级的开发技巧，以及独立做项目的“技术”能力。
  • 可以得到助教的线上协助。
  • 求职技巧辅导

线下 5 周版

  • 以 Growth 的观点打造 MVP 产品
  • 世界黑客松首奖得主指导，如何在五周之内协作打造商业团队五个月开发等级产品。
  • 高阶全栈产品开发技巧
  • 高阶程序员的协作、设计、部署、调优
  • GrowthHack 你的产品
  • 企业合作优先录取

这个营队似乎价格稍高，能说说你们的优势吗？

旧金山的 Hackreactor

  • 课程至少是 16 周（线上 + 线下）以上，教的只是“技术上的全栈”
  • 价格是 $17,780 USD（¥122,026 RMB )
  • 而初期订金就至少要 $2,000 USD ( ¥13,700 RMB )。

线上的 LaunchSchool

  • 采订阅制 $199 USD / month
  • 至少要 6 - 12 个月以上学习才能毕业求职
  • 价格需花费 ¥8,190 - ¥16,380 RMB。而且这当中有小段时间（1-2个月）甚至必须要辞职在家专心学习。

全栈营

  • 全栈营的线上 Program 是以三个月为一期
  • 目的是让学生在经过三个月训练后（很可能可以不用辞职，看个人天赋）可以到达求职阶段
  • 这一年内可以重复上课

平日可用时间、学习效率远比费用（其实比上述两间远来得划算）来的珍贵，你难道不觉得吗？

学习这门课需要哪些基础？

我们本以为，具备以下背景的人会学得快许多（至少其一）： 

  • 较好的英文水平 
  • 自己捣鼓过 HTML / CSS / JavaScript 
  • 至少使用过一门编程语言（有基本能力即可） 
  • 良好的中文写作能力 
  • 日常操作系统环境是 Mac OSX

然而通过对线下课和线上课学员的观察，我们发现，90%的同学都是英文一般、编程零基础、没有中文写作能力、上了全栈营才第一次接触 Mac OSX，他们却一样能学得很好。

请问这个课需要怎么样的英文水平？

客观的说，编程世界常用的英文单词，只有200多个，所以，我们并不要求你是英文大牛。
同时，如果你具备以下程度，学习会顺利不少;如果不具备，在我们的学习过程中也可以慢慢培养出来：

  • 习惯使用 Google 查找资料，并能辨认找到的资讯是否有价值 
  • 可以快速地略读英文材料

需要全职学习吗？

本课程为线上进行，你可以自行安排适合的学习时间，建议每天花 2 个小时进行学习。

第二期课程什么时候上线？

  • 4 月 3 日之前是课前作业练习时间，请查看学习中心的《VIP 入学手册》并完成 Rails第一课、第二课、第三课的作业
  • 4 月 3 日开始授课

线上课程有助教答疑吗？

  • “付费会员”有助教技术答疑
  • “付费会员”会被邀请进入线上沟通软件，每天会有固定的线上技术答疑时段

课程除了线上教材，会有视频、音频吗？

我们会根据课程进度采用视频直播讲解课程内容以及答疑，错过直播的可以查看回放。

这个课会有时间限制吗？

购买 VIP 会员后，所有的线上付费课程均可以在会员有效期内（购买之日起至 2017 年 12 月 31 日）不限次数学习

学编程一定得使用 Mac 吗？不能用 Windows 吗?

  • 我们教材均以 Mac 系统为基础编写
  • 若你真心要将编程当作一门认真的事，那么一定得使用 Mac 编程
  • Mac 上有着开源世界所有最好程序员提供的各式工具以及最佳配置，可以让你至少躲掉一百个以上的坑

买 Mac 有什么需要注意的？

内存 8G (或8G以上)，否则未来在跑程序可能会发生无法执行的情况。

除了买 Mac 之外，学习编程还需要准备什么？

这里有几样东西，我们也希望你额外准备。 

  • 一个外接屏幕（尺寸越大越好，在看代码的时候会方便很多） 
  • 一把好的机械式键盘（提高输入效率）
  • 一个“可以接触墙外世界的网络帐号”（你懂的...） 
  • 好的网速（据学员反映，初学编程最大的挫折是下载安装环境太慢...）
  • Gmail 帐号

如何购买 VIP ？

注册/登录后在学习中心点击“开通 VIP”并提交订单，付款完成即可成为 VIP 会员。

有哪些付款方式？

  • 支付宝支付
  • Paypal支付（仅支持美元收付，建议海外用户使用）
  • 可以请朋友对上述付款方式进行代付

可以分期付款吗？

可以，在使用支付宝／Paypal支付的时候，选择信用卡支付，之后再对信用卡账单进行分期即可。

可以开发票吗？

可以开普通发票，请和在线客服联系。

可以退款吗？

  • 本网站在线课程/VIP会员为虚拟商品，原则上不提供退款服务
  • 若您个人有特殊情况，自付款成功之日起，七天之内（含七天）可以申请退款；付款成功超过七天之后请恕无法受理退款

我已经进行支付了，但还没有升级成 VIP ？

点击网页右下角在线客服，向客服提供：

  • 网站订单号
  • 网站注册邮箱
  • 支付宝／Paypal订单详情页截图

我们的客服人员会在核实资料后帮您升级。

我们会根据课程进度采用视频直播讲解课程内容以及答疑，错过直播的可以查看回放。

这个课会有时间限制吗？

购买 VIP 会员后，所有的线上付费课程均可以在会员有效期内（购买之日起至 2017 年 12 月 31 日）不限次数学习

学编程一定得使用 Mac 吗？不能用 Windows 吗?

  • 我们教材均以 Mac 系统为基础编写
  • 若你真心要将编程当作一门认真的事，那么一定得使用 Mac 编程
  • Mac 上有着开源世界所有最好程序员提供的各式工具以及最佳配置，可以让你至少躲掉一百个以上的坑

买 Mac 有什么需要注意的？

内存 8G (或8G以上)，否则未来在跑程序可能会发生无法执行的情况。

除了买 Mac 之外，学习编程还需要准备什么？

这里有几样东西，我们也希望你额外准备。 

  • 一个外接屏幕（尺寸越大越好，在看代码的时候会方便很多） 
  • 一把好的机械式键盘（提高输入效率）
  • 一个“可以接触墙外世界的网络帐号”（你懂的...） 
  • 好的网速（据学员反映，初学编程最大的挫折是下载安装环境太慢...）
  • Gmail 帐号
