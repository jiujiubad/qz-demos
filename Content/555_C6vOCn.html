7-4 Git

  • 程序员用来保存项目代码的「时光机」。

常见 Git 的指令

  • git init  （把专案变成一个被 git 记录的专案）
  • git add README   ( 把某个文件加入 git 的控制）
  • git add . （把当前目录加入 git 的控制）
  • git commit -m “信息”  （把已经加入控制的文件，正式记录到变更中）

进阶 Git 指令

  • git remote add origin  git://github.com/xxx.git
    • 新增一个远程的仓库，命名为 origin。origin 的位置指向 Github
  • git push origin master
    • 推送本地的 master 分支到远程的 origin 仓库
  • git push -u origin master
    • 预设以后都是推到远程 origin 仓库的 master
  • git status
    • 当前代码追踪的状态。红色代表被变更的内容，绿色代表现在修改的内容
  • git pull
    • 拉下远程仓库的变更
  • git checkout -b ch01
    • 新增一个名为“ch01”的分支
  • git checkout ch01
    • 切换到 “ch01” 分支
