作业 - 作业二
Enumerable module:
响应新的副本：
collect：返回新的数组，新数组元素是原来数组元素经过块处理后的结果。
drop:删除对象中前n个元素，返回剩下的元素。
drop_while:删除元素，知道碰到的元素在块中的结果为false或nil，即删除满足块中条件中元素，直到遇到第一个不满足条件的元素，便停止运行
find:返回满足条件的第一个元素
find_all:返回满足条件的所有元素

array class中成对出现的方法：(带有！为in-place修改方法)
collect：返回经过处理后的新的数组
compact:删除数组中的nil元素
flatten：将嵌套的数组展开
map：与collect类似
reject：删除满足条件的元素，返回剩下的元素
reverse:所有元素反向排序
rotate：反转，以指定元素为界限，让指定元素前面的元素放到数组的后面
slice：返回指定的元素
sort：排序
uniq：删除重复的元素

hash中成对出现的方法：
merge：将两个散列作并集操作
reject：删除
select：获取元素
transform_values
compact:删除值为nil的元素

成对的 
arr.rotate 和 arr.rotate! 旋转，后面接着元素的位置来表示谁排在第一位（默认为1），然后依次排序
arr.reverse 和 arr.reverse! 反向排序
arr.select { |item| block } 和 arr.select! { |item| block }筛选，在所有的元素中寻找block中存在的元素
arr.shuffle 和 arr.shuffle! 随机排列
arr.shuffle(random: rng) 和 arr.shuffle！(random: rng) rng参数用作随机数生成器
arr.sort 和 arr.sort! 从小到大按顺序排列
arr.sort { |a, b| block } 和 arr.sort! { |a, b| block } 可以指定反向从大到小排列
arr.uniq 和 arr.uniq! 去掉重复
arr.uniq { |item| ... } 指定除去指定位置的重复元素
hash.compact 和 hash.compact! 精简哈希值，除去空值

响应新的副本：
arr.sample 随机抽样，后面接数字表示抽几个元素，默认为1
以上没有！号的都是响应新的副本

修改方法：
arr.shift 移除元素，后面可以接数字表示几个元素，默认为1，数字表示多少个，从第一个开始移除
arr.slice! 移除元素，后面可以接数字表示指定位置的元素，没有默认值，数字表示位置,可以删除多个指定元素
arr.unshift 增加元素，后面小括号直接加元素，可以增加多个元素，直接到最前面
hash.clear 删除所有哈希值
以上有！号的都是修改方法
